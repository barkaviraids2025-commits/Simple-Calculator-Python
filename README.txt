Project Name: Simple Calculator – Python

Description:
This project is a basic calculator program written in Python. 
It accepts two numbers and an operator (+, -, *, /) from the user 
and displays the result.

Objective:
- Learn Python basics
- Understand input and output functions
- Apply conditional statements
- Handle arithmetic operations
- Learn how to structure a simple project

Features:
1. Addition
2. Subtraction
3. Multiplication
4. Division (with zero-division check)
5. User-friendly input system

Technologies Used:
- Python 3

How to Run:
1. Open the project folder
2. Click the address bar at the top
3. Type "cmd" and press Enter
4. Type the following command:
   python calculator.py

Sample Input/Output:
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): *
Result: 50

Project Structure:
- calculator.py        → main program
- README.txt           → explanation and instructions
